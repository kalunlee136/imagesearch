Image Search Abstraction Layer Project

Goal: To create an abstract image search app that allows one to input parameters to search for images/pics

Example api call: https://www.example.com/api/imagesearch/?search=lolcats

Example output: {"data":[{"link":"http://i.imgur.com/Vw9clVQ.jpg","title":"LOLcats","account_url":"downess"}]}



Allows for pagination as well

Example api call: https://www.example.com/api/imagesearch/?search=lolcats&offset=1

Example output: {"data":[{"link":"http://i.imgur.com/d1PAOZU.jpg","title":"The Net is not educational. Just porn and lolcats. Books are still boss.","account_url":"Lassannn"},{"link":"http://i.imgur.com/FqJTc9n.png","title":"title","account_url":"ElishaShaddock1"},{"link":"http://i.imgur.com/IQs5vr1.jpg","title":"Kitty moms have to put up with the same stuff ","account_url":null}]}

To check previous searched queries:

Example api call: https://www.example.com/api/imagesearch/last
